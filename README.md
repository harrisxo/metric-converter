# Metric Converter

Ein kleiner Metric/Imperial-Umrechner für Länge, Volumen und Gewicht.

## Funktionen
- Meter ⇄ Feet
- Liter ⇄ Gallons (US)
- Kilogramm ⇄ Pounds

## Nutzung
1. Zahl eingeben
2. Auf **Convert** klicken
